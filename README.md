# Multi-Window-OpenCV

A Python project utilizing OpenCV to create and manage multiple windows for image processing.

## Introduction

This project demonstrates how to create and manage multiple windows using the OpenCV library in Python. By leveraging the capabilities of OpenCV, users can visualize and process images in different windows simultaneously.

## Features

- **Multi-Window Display:**
  Create multiple windows to display different images or processed results simultaneously.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ChetanaVarahachalam/Multi-Window-OpenCV.git
   ```

2. Navigate to the project directory:

   ```bash
   cd multi-window-opencv
   ```

3. Install dependencies:

   ```bash
    python -m pip install --upgrade opencv-python
   ```

## Usage

Run the main script to launch the multi-window OpenCV application:

```bash
python main.py
```


## Dependencies

- Python 3.x
- OpenCV
